# 医学知识库系统

一个基于 React 和 Ant Design 的医学知识检索系统。

## 功能特点

- 🔍 实时搜索功能
- 📚 疾病、药品、检查等多类别信息
- 💊 详细的医学信息展示
- 📱 响应式设计，支持移动端
- ⚡ 快速的搜索响应

## 技术栈

- React
- Ant Design
- React Router
- CSS3

## 快速开始

1. 克隆项目
```bash
git clone [repository-url]
cd medical-knowledge-base
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm start
```

4. 打开浏览器访问 http://localhost:3000

## 项目结构

```
medical-knowledge-base/
```
